Tools Development

For this module, my goal was to design and create a tool for Unreal Engine which could be used to speed up the development of a game. It was my first time making an editor tool and so I made a Dungeon Generator tool using a GUI to arrange the layout on a square grid and select the rooms.

Key Features

  • Room presets: Room presets can be applied using the GUI to select a specific room type for the selected square.
  • Corridor Allocation: Alongside selecting the rooms, corridors could be selected in the GUI to connect rooms together.
  • Corridor length: This lets the user create distance between each room and thus make the corridors longer.
