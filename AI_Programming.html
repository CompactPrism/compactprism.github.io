AI Programming Fundamentals

This module took me through fundamental concepts of AI such as steering behaviours, pathfinding, and decision making. There was no main project for this module as instead, I made a single Unity project containing demonstrations of the concepts.

Concepts Explored

Part 1 - Steering Behaviours

  • Seek
  • Wander
  • Collision Avoidance
  • Flee
  • Arrive
  • Pursuit
  • Evade
  • Group Movement

Part 2 - Pathfinding Algorithms

  • Dijkstra
  • A*

Part 3 - Decision Making

  • Finite State Machine
  • Fuzzy State Machine
  • Behaviour Tree

Implementation Details

The Unity project in this module had several preset scenes provided by the university and in each scene, I needed to apply these concepts to the character. I successfully managed at the end to combine a Behaviour Tree with steering behaviours to help a character navigate a specific scenario.
